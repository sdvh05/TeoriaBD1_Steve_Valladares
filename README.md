# TeoriaBD1_Steve_Valladares
Sistema de Presupuesto Personal
Descripción

Este proyecto consiste en el desarrollo de un Sistema de Presupuesto Personal que permite gestionar ingresos, gastos, ahorros y obligaciones mensuales, 
facilitando el control y análisis de las finanzas personales del usuario.

La aplicación está diseñada para registrar transacciones, organizar la información por categorías y generar reportes que apoyen la toma de decisiones financieras.

Tecnologías utilizadas:
  Lenguaje de programación: Java
  Base de datos: Oracle Database

Características principales

  Gestión de usuarios
  Registro de ingresos, gastos y ahorros
  Clasificación por categorías y subcategorías
  Control de presupuestos mensuales
  Consultas y reportes financieros

Arquitectura
El sistema sigue una arquitectura de capas:

  Capa de lógica: desarrollada en Java
  Capa Visual: desarrollada en Java
  Base de Datos: implementada en Oracle

Propósito
  Este proyecto fue desarrollado con fines académicos para aplicar conceptos de programación en Java y diseño y manejo de bases de datos relacionales en Oracle.
